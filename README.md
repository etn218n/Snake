## Installation

For Ubuntu:

Install ncurse library
```
sudo apt-get install libncurses5-dev libncursesw5-dev
```

Change directory to cloned repo & use make to run compilation commands
```
cd ./Snake
make
```

Run game
```
./Snake
```
